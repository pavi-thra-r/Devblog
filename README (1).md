# 📝 Blog Web Application

A modern **Blog Web Application** built using **React + Vite** that allows users to create, view, and manage blog posts.  
The application includes authentication pages, a blog editor, and structured pages for user interaction.

---

# 🚀 Features

- User Registration
- User Login
- Create Blog Posts
- Blog Editor Interface
- View Blog Posts
- Modern React UI
- Component-based Architecture
- Fast development using **Vite**

# 🖥️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/blog-project.git
```

Navigate into the project directory:

```bash
cd blog-project
```

Install dependencies:

```bash
npm install
```

---

# ▶️ Run the Project

Start the development server:

```bash
npm run dev
```

The application will start on:

```
http://localhost:5173
```

# 📄 Pages

### 🏠 Home Page
Displays the main landing interface for the blog platform.

### 🔐 Login Page
Allows existing users to log in.

### 📝 Register Page
Allows new users to create an account.

### 📚 Blog Page
Displays blog posts.

### ✍️ Blog Editor
Allows users to write and publish blog content.
